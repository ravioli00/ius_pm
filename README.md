# Information Uniqueness Score

> *Do prediction markets contain information that financial markets don't already know?*
> A frozen-protocol test on 16 markets, the methodology that produced the test, and
> the synthetic Monte Carlo that anchors the empirical result.

[**📄 Paper**](paper/main.pdf) &nbsp;·&nbsp; [**📋 Protocol**](protocol/protocol.md) &nbsp;·&nbsp; [**📊 Results**](outputs/) &nbsp;·&nbsp; [**🧪 Tests**](tests/)

---

The **Information Uniqueness Score (IUS)** is an information-theoretic ratio
that predicts when a prediction market should causally lead financial markets:
when its information about the underlying event is non-redundant conditional
on observable financial variables. Its computable proxy, the **Information
Leadership Score (ILS)**, is built from incremental cross-predictive $R^2$.

A frozen analysis plan on 16 prediction markets ($\$7.8\text{B}+$ Polymarket
volume, $22\text{,}000+$ hourly observations) tested whether IUS rank tracks
empirical leadership across the panel. The headline:

| Test | $\rho(\mathrm{ILS}, L)$ | $p$ |
|---|---:|---:|
| **Pre-specified, full panel** ($n = 16$) | $-0.046$ | $0.868$ |
| Synthetic baseline at $n = 16$, $\beta = 0.6$ | $+0.96$ | $< 10^{-4}$ |
| Long-panel subset, Granger-$F$ leadership ($n = 5$) | $+0.95$ | $0.034$ |

The framework's prediction is **not supported as pre-specified**, but the
synthetic recovery rules out a methodology artifact and a post-hoc test on
the markets where the methodology has enough data ($n_\text{used} \ge 800$)
shows the predicted positive direction across all five leadership
operationalisations.

## What's tested

The package code in `ius/` is covered by 37 pytest cases:

- **`test_simulate.py`** — DGP shape and determinism; leader / follower /
  independent direction recovery; regime-switch sign flip; heavy-tail
  kurtosis vs Gaussian; latent-confounder cross-correlation; raises on
  unknown DGP.
- **`test_ils.py`** — strong leader gives $\mathrm{ILS} > 1$ and strong
  follower gives $\mathrm{ILS} < 1$ in $\ge 12 / 15$ seeds; alias resolution
  for ETF-proxied channels (DXY ↔ UUP, etc.); short-panel and missing-channel
  edge cases; bootstrap CI brackets the point estimate.
- **`test_leadership.py`** — leader gives $L > 0.7$, follower $L < 0.3$;
  independent DGP gives $0.2 \le L \le 0.8$; constant input gives neutral
  $L = 0.5$; below `min_n`, returns NaN; lag sensitivity 1, 2, 3.
- **`test_stats.py`** — perfectly correlated and anticorrelated data give the
  expected one-/two-sided $p$; random data $p$ concentrates uniform; same
  RNG → same $p$; input-validation paths raise.
- **`test_pipeline.py`** — end-to-end recovery at realistic and empirical
  panel sizes; partitioned leader-vs-follower panel sign recovery.
