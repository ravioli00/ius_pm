# Information Uniqueness Score: A Pre-Registered Null on a 16-Market Prediction-Market Benchmark

This repository accompanies *The Information Uniqueness Score: A Pre-Registered Null on a 16-Market Prediction-Market Benchmark* (Sayahpour, 2026).

The paper proposes the **Information Uniqueness Score (IUS)**, an information-theoretic ratio that predicts whether a prediction market should causally lead or follow financial markets, and tests this prediction with a pre-registered protocol on a 16-market panel. The pre-registered Spearman $\rho(\mathrm{ILS}, L) = -0.046$ at permutation $p = 0.868$, **the framework's prediction is not supported on the panel as pre-registered.** Synthetic Monte Carlo recovery shows the methodology works at the linear-Gaussian baseline ($\rho = +0.96$ at $n = 16$, permutation $p < 10^{-4}$), so the empirical null is informative rather than a power artifact.

The full paper is at [`paper/main.pdf`](paper/main.pdf).

## Install

```bash
git clone https://github.com/<user>/ius-prediction-markets
cd ius-prediction-markets
python -m venv .venv && source .venv/bin/activate
pip install -e ".[test]"
```

## Run the test suite

```bash
pytest -v
```

The 37 tests cover the IUS estimator (known-DGP recovery, alias handling, bootstrap), the Granger-F leadership score (sign recovery, edge cases, lag sensitivity), the synthetic data-generating process (DGP-direction validation, regime-switch sign flip, heavy-tail kurtosis, latent-confounder cross-correlation, seed determinism), the permutation-Spearman test (calibration, determinism, input validation), and an end-to-end pipeline check.

## Reproduce paper results

```bash
# Synthetic Monte Carlo (Appendix D, Tables 5-7, Figure 2): ~3-5 minutes
python experiments/synthetic_validation.py

# Empirical subset robustness (Section 5.5, Table 4): ~1 minute
python experiments/empirical_robustness.py

# Direct test of Mechanism (1) on long-panel subset (Section 6, Table 8): ~30 seconds
python experiments/long_panel_subset.py
```

Outputs land in `outputs/`. To regenerate Figure 1 (IUS scatter) and Figure 2 (synthetic recovery curves), see `figures/`.

## Headline results

| | $\rho(\mathrm{ILS}, L)$ | permutation $p$ | $n$ |
|---|---|---|---|
| **Pre-registered, full panel** | $-0.046$ | $0.868$ (two-sided) | $16$ |
| Long-panel subset, pre-registered $L$ | $+0.20$ | $0.395$ (one-sided) | $5$ |
| Long-panel subset, Granger-F $L$ | $+0.949$ | $\mathbf{0.034}$ (one-sided) | $5$ |
| Long-panel subset, PCMCI+ directed-edge ratio | $+0.894$ | $\mathbf{0.048}$ (one-sided) | $5$ |
| Synthetic baseline at $n = 16$, $\beta = 0.6$ | $+0.96$ | $< 10^{-4}$ | $16$ markets simulated |
| Synthetic, regime-switch DGP, $n = 30$ | $+0.58$ | $0.0007$ | ground-truth recovery collapses |
| Synthetic, strong latent confounder, $n = 30$ | $+0.67$ | $0.0002$ | recovery degrades |

The pre-registered null is the headline. The long-panel subset post-hoc is reported as a direct test of Mechanism (1) (panel-length confounding) and is underpowered at $n = 5$; replication on a panel pre-registered to satisfy $n_\text{used} \ge 800$ is the natural next test. The synthetic results show the methodology is sound at baseline and identify regime-switching and strong latent confounders as the most likely DGP-violation explanations of the empirical null.
